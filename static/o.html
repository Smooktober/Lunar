Options

Here, you can change multiple options to make Lunar truly yours, such as your desired backend, cloaking options, visual preferences, and more.


TAB CLOAKER

Make Lunar look like any site you'd like! Change the name and icon that appears in the tab name as well as your search history.


A:B CLOAK

A:B Cloaking opens your desired site in a new window that is hidden to various web filters. It also omits sites you visit from your search history!



BACKEND SELECTION

Choose between multiple different backends, each with different advantages.


Ultraviolet

The current standard for web proxy technologies. Supports almost every site with quick speeds.



Dynamic (Currently Down)

Currently in Public Beta. Supports a wide range of sites with blazing fast speeds.



THEME SELECTION

Customize the appearance of Lunar to fit your visual needs.



SEARCH ENGINE SELECTION

Choose from a wide variety of search engines, or specify your own. Valid search engine URLs will start with "https://" and end in the following format: "?" + (letter) + "=".



EMERGENCY REDIRECT

In the event you need to quickly change the current page to a different site, you can do so by selecting a URL and key below. When the selected key is pressed on your keyboard, it will redirect you to a site of your choosing. By default, pressing the backslash key ("\") will redirect you to Google.

TOOLS

These options are used to fix client-side bugs and errors with Lunar. Keep this in mind prior to requesting support as most issues can be easily fixed yourself.


RESET SITE

Pressing this button clears all data stored by the site. It re-registers all service workers, clears all cookies, and deletes LocalStorage data, essentially resetting the site to it's default settings. This can be used as an easy fix for most basic issues with Lunar.
